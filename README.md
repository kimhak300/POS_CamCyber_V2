## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

# migrate
```bash
$ npm run migrate
```

# seed
```bash
$ npm run seeder
```
